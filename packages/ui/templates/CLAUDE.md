# Templates

## Purpose

Page layout patterns that define structure without content. Templates are shells that pages fill.

## Layout Focus

Templates define grid, flex, and spacing patterns. They position slots where content goes.

## Props

Accept children or render props for content slots. Don't accept data directly.

## Responsiveness

Handle responsive layout changes. Mobile, tablet, desktop variations defined here.

## Examples

Dashboard layouts, settings layouts, detail page layouts, list/detail split views.
